Sobre este Proyecto

Este proyecto está diseñado para ayudar a los nuevos contribuidores en Git y GitHub a familiarizarse con el proceso de colaboración en proyectos de código abierto.

Aprender a contribuir a repositorios es un paso importante, y nuestro objetivo es que puedas hacerlo de manera fácil y sin presión. ¡Aquí te guiaremos para que hagas tus primeras contribuciones!

Instrucciones para Contribuir

  1. Bifurca (*Fork*) este repositorio.
  2. Abre un Codespaces desde tu fork.
  3. Crea una nueva rama con tu nombre.
  4. Crea una carpeta en la ruta /Alumnos/Listado y añade un archivo con un enlace a tu perfil de GitHub.
  5. Confirma y haz push de tus cambios.
  6. Crea una pull request para que tus cambios sean revisados.

Consulta las instrucciones detalladas en el archivo README.md de este repositorio

¿Qué es Git y GitHub?

Git es un sistema de control de versiones que te permite gestionar el historial de cambios en tu código. GitHub es una plataforma donde puedes almacenar tus repositorios y colaborar con otros.

Aprenderás sobre forks, branches, commits, pull requests, y mucho más a lo largo de este proyecto.
